Space Invaders
==============

A basic Space Invaders clone.

Written in Summer 2012 by Eric Turner

Controls
--------

Move:              Left and Right arrow keys

Fire:              Spacebar

Pause/Unpause:     p

Unofficial Controls (aka Cheats)
--------------------------------

Auto-skip level:   b

drop a powerup:    c
